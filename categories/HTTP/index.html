localhost、127.0.0.1和0.0.0.0

2019-07-29
IP
localhost 127.0.0.1 本机IP
网络 不联网 不联网 联网
传输 不使用网卡，不受防火墙和网卡限制 网卡传输，受防火墙和网卡限制 网卡传输，受防火墙和网卡限制
访问 本机访问 本机访问 本机或外部访问

1、localhost，域名，可以被配制成任意ip地址，通常情况下指向127.0.0.1（ipv4，通过本机host文件，将localhost自动解析为127.0.0.1，可修改）和[::1]（ipv6）；

2、127.0.0.1，127.0.0.1:8整个都是环回地址，通常分配给loopback接口，用来测试本机的TCP/IP协议栈，发往这段A类地址数据包不会出网卡，网络设备不会对其做路由，linux系统中此接口名称为lo；

3、Any（0.0.0.0）地址被称为通配地址，使用此IP地址可以建立的服务器，IP为本地主机上所有的IP地址；

如果一个主机有两个IP地址：192.168.1.1和10.1.2.1，若该主机上的一个服务监听地址为0.0.0.0，则通过两个ip都能访问该服务；

4、本机IP，本机有三块网卡，一块网卡叫做loopback（虚拟网卡），一块叫做ethernet（有线网卡），另一块叫做wlan（无线网卡）。本机IP就是真实网卡的IP，具体来说有线无线各一个，127.0.0.1对应looback虚拟网卡。
